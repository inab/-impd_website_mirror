Plan de Trabajo

  1. Inicio
  2. Quiénes somos
  3. Plan de Trabajo

El plan de trabajo de IMPaCT-Data está detallado en el plan estratégico de IMPaCT, disponible en la web del Instituto Carlos III (ISCIII).

El trabajo está distribuido en 6 paquetes de trabajo (WP por sus siglas en inglés - Work package). Los detalles de los entregables de los paquetes de trabajo están accesibles en las entregables del proyecto.

WP1. COORDINACIÓN

Coordinación del proyecto.

Tareas:

  • Coordinación con los otros dos programas de IMPaCT.
  • Coordinación con los servicios de salud de las CC.AA.
  • Coordinación y gestión interna del programa.
  • Comunicación y difusión de las actividades del programa.
  • Coordinar la participación de las entidades asociadas en las actividades del programa.
  • Formación específica.

WP2. INFRAESTRUCTURA

Infraestructura computación en la nube para la gestión e integración de datos.

Tareas:

  • Implementación inicial y evaluación de un sistema de computación para datos biomédicos asociados a IMPaCT.
  • Adopción e implantación de protocolos para el uso de contenedores software y flujos de trabajo siguiendo las recomendaciones de ELIXIR, EOSC-Life y GA4GH.

WP3. DATOS GENÓMICOS

Implementación de sistemas para el análisis, integración e interpretación de datos genómicos.

Tareas:

  • Implementación de sistemas de almacenamiento de información genómica primaria mediante instancias Federadas de EGA.
  • Adaptación, instalación y uso de software de código abierto para el análisis e integración de distintas fuentes de datos.

WP4. DATOS CLÍNICOS

Implementación de sistemas para el análisis, integración e interpretación de datos clínicos incluyendo imágenes.

Tareas:

  • Adaptación, instalación y uso de software de código abierto para la extracción de variables a partir del HCE.
  • Adaptación, instalación y uso de herramientas de software de código abierto para el seguimiento y validación de los procesos de extracción y tratamiento de los datos de imagen médica.

WP5. INTEGRACIÓN

Diseño e implementación inicial de los sistemas de interconexión entre datos genómicos, imagen médica y médicos.

Tareas:

  • Integración de datos genómicos y radiómicos con datos médicos estructurados para su uso secundario.
  • Implantación de mecanismos para la identificación automática de información genómica relevante, o contenido de las imágenes médicas que den respuestas anonimizadas al estilo beacon.

WP6. CASOS DE USO

Casos de uso para la aplicación de la metodología y evaluación de su calidad.

Tareas:

  • Desarrollo de los casos de uso alineados con IMPaCT.
  • Evaluación de aspectos de seguridad en el manejo de datos sensibles, como procesos de pseudonimización, accesibilidad, trazabilidad y privacidad de los datos.
