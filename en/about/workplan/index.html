Work Plan

  4. Work Plan

The IMPaCT-Data’s work plan is detailed in the IMPaCT Strategic Plan, available at the Carlos III Institute (ISCIII) website.

The work is distributed in 6 work packages (WP for its acronym in English - Work package). Details of deliverables work packages are accessible in theproject deliverables.

WP1. COORDINATION

Project management.

Tasks:

  • Coordination with the other two IMPaCT programs.
  • Coordination with the health services of the Autonomous Communities.
  • Coordination and internal management of the program.
  • Communication and dissemination of program activities.
  • Coordinate the participation of associated entities in program activities.
  • Specific training.

WP2. INFRASTRUCTURE

Cloud computing infrastructure for data management and integration.

Tasks:

  • Initial implementation and evaluation of a computer system for biomedical data associated with IMPaCT.
  • Adoption and implementation of protocols for the use of software containers and workflows following the recommendations of ELIXIR, EOSC-Life and GA4GH.

WP3. GENOMIC DATA

Implementation of systems for the analysis, integration and interpretation of genomic data.

Tasks:

  • Implementation of primary genomic information storage systems through Federated instances of EGA.
  • Adaptation, installation and use of open source software for the analysis and integration of different data sources.

WP4. CLINICAL DATA

Implementation of systems for the analysis, integration and interpretation of clinical data including images.

Tasks:

  • Adaptation, installation and use of open source software to extract variables from the HCE.
  • Adaptation, installation and use of open source software tools for the monitoring and validation of medical image data extraction and treatment processes.

WP5. INTEGRATION

Design and initial implementation of interconnection systems between genomic data, medical imaging, and clinical data.

Tasks:

  • Integration of genomic and radiomic data with structured medical data for secondary use.
  • Implementation of mechanisms for the automatic identification of relevant genomic information, or medical imaging content that give anonymized responses in the beacon style.

WP6. USE CASES

Use cases for the application of the methodology and evaluation of its quality.

Tasks:

  • Development of use cases aligned with IMPaCT.
  • Assessment of security aspects in the handling of sensitive data, such as pseudonymization processes, accessibility, traceability and data privacy.
