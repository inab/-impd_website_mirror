Roadmap

  1. Home
  4. Roadmap

December 2021 - Technical Requirements

* The technical requirements for the different technologies needed are identified: * Technical requirements for the deployment of a local instance of EGA * Technical requirements for Beacon deployment for data discovery *State of Technology Practice for IMPaCT-Data Cloud Design * Definition of use cases

June 2022 - Technical Demonstrator and State-of-the-Art

* Demonstration of the IMPaCT-Data federated cloud, deployed on multiple nodes and providing: * controlled user access * access to public data * test analysis workflow * State of the art of data annotation: * State of the art in clinical history data annotation * State of the art in image data annotation * State of the art in genomic data annotation * Clinical history, image and genomic data integration techniques * Good practice guides for the development of research software components * Recommendations on security aspects in the management of sensitive data * Definition of interfaces EGA community instances

September 2022 - Tools Catalogues

Different software tools are compiled for the analysis of genomic, clinical history and image data: * Catalog of software tools * Catalog of virtualized software tools to be executed * Workflow catalog

Diciembre 2022 - IMPaCT-Data Cloud (access to data)

* Initial version of the IMPaCT-Data cloud including: * access to public data and controlled access * tools and workflow for the analysis of genomic data, clinical history and image (not integrated) * Definition of the use case evaluation strategy

June 2023 - IMPaCT-Data Cloud (access to integrated data)

* Consolidated version of IMPaCT-Data cloud to be tested by project members * Demonstrator of the integration of genomic, clinical history and image data * Use cases in operation

October 2023 - IMPaCT-Data Cloud (Proof-of-Concept)

* Final version of the IMPaCT-Data cloud proof of concept

December 2023 - Project Finalisation

At the end of the project, deliverables have been planned with the final report of all the work packages.
